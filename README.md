# Paper_Review and Practice

This repository personal reviews and practice for deep learning research papers.  
The focus is on influential and work in computer vision, natural language processing, and generative models.

Each paper is accompanied by one or more of the following:
- presentation notes
- Written summary (PDF or Markdown)
- Minimal code or notion linnk

If you have questions or suggestions, please open an issue in this repository.
